Cài Đặt EmailJS
Hướng dẫn cài đặt EmailJS:
1. Đăng ký miễn phí tại emailjs.com
2. Vào Email Services → Add Service → chọn Gmail → lấy Service ID
3. Vào Email Templates → Create Template → lấy Template ID
4. Vào Account → lấy Public Key
5. Điền các thông tin bên dưới và nhấn Lưu
